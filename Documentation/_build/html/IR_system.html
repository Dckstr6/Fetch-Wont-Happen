  • string2 – string to which another string has to be edited to

preprocess(text)

This function goes through the text of a document and tokenizes it, removes stopwords, lemmatizes it and returns a string that is free of stopwords and the rest of the tokens lemmatized.

Parameters

text – the text in the document passed as string

preprocess_text_files()

This function reads all the data files and send the text in it to the preprocess() function for further processing.

process_spelling_mistake(word)

This function when given a word that is considered to have wrong spelling uses the levenshteinDistanceDP() function to find the edit distances from all the recognised words in the vocabulary and returns the best suited word (least edit distance).

Parameters

word – The misspelled word for which a word must be found in the vocabulary that is close to the original spelling

process_user_query(user_input)

This function is used to process the query given by the user. It is first sent to infix_to_postfix() to get back the postfix stack after which a for loop evaluates the postfix boolean expression to get the resultant documents.

Parameters

user_input – the query that the user gives.
